Keypress

Console

Keypress is an esoteric programming language. It listens for keypress events on the document to trigger commands. There is no code editor, all code is executed live.


Click on the console above to begin coding, or read through the documentation below:

Key Command
Delete Reset all values
Backspace Clear the console
Right Arrow Move the pointer to the right
Left Arrow Move the pointer to the Left
Up Arrow Increment the value of the current cell by 1
Down Arrow Decrement the value of the current cell by 1
Enter Output the ASCII character that corresponds to the current value of the cell
0 Set value of current cell to 0
c Copies value of current cell into "clipboard"
v Adds value of "clipboard" to current cell
